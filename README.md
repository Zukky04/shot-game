ファイル形式の互換性: AQUOS SH-02Lの「ドコモ電話帳」アプリで連絡先をインポートする際は、vCard形式（.vcf）のファイルを使用する必要があります。CSV形式などの他の形式のファイルを直接インポートすることはできません。
